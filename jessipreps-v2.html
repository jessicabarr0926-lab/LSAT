📖 Reading Comprehension

Passage Map
Label each paragraph's purpose. Track viewpoint shifts. Separate the author's view from others' views. Mark contrast words: but, however, yet, while, although.

Main Point
Umbrella test: does the answer cover the whole passage? Avoid answers that are too narrow (just one paragraph) or too broad (claims more than is there).

Inference
The 100% Rule: the answer must be true whenever the passage is true. No outside knowledge. Eliminate extreme language (always, never, all, only).

Author's Attitude
Find the evidence word. Can you point to a sentence? Mild descriptors beat extreme ones (critical > dismissive, supportive > enthusiastic).

⚖️ Logical Reasoning

Flaws
Find conclusion → find evidence → say what the argument assumes → match the gap to the answer. Common flaws: correlation≠causation, treating tendency as certainty, necessary vs sufficient confusion.

Assumptions (Negation Technique)
Negate each answer choice. If the conclusion collapses → that answer IS the assumption. The unstated bridge must be required, not just helpful.

Strengthen / Weaken
Identify the causal or logical bridge. Predict what would help or hurt it. Avoid choices that only discuss the topic without affecting the argument's logic.

Conditional Logic
Translate trigger words into arrows (→). Write the contrapositive immediately (NOT B → NOT A). Never reverse (A→B does not mean B→A). Chain linked rules.

⏱️ Pacing

Know the task within 20 seconds. If the structure is unclear — skip it. Return with a cleaner eye. Every question is worth the same point.

LR: ~1 min 25 sec per question. RC: read 3–4 min per passage, ~1 min per question. Flag hard questions and keep moving — running out of time costs more than getting one question wrong.
